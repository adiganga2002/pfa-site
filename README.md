# Adrian Gânga PFA — site (RO/EN)

Site static (HTML/CSS/JS) pregătit pentru GitHub Pages.
Include:
- comutator limbă RO/EN (persistă în localStorage)
- pagină separată „Servicii”
- pagină separată „Confidențialitate (GDPR)”

## Publicare pe GitHub Pages
1. Creează un repo (ex: `pfa-site`)
2. Copiază fișierele din acest folder în root-ul repo-ului
3. GitHub → Settings → Pages → Deploy from a branch
4. Branch: `main`, Folder: `/ (root)`
5. Deschide URL-ul generat de GitHub Pages

## Personalizare
- În `index.html` schimbă email/telefon și link-urile GitHub/LinkedIn.
- Dacă adaugi formulare sau analytics, actualizează `privacy.html`.
